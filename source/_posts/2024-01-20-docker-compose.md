# Docker compose
## 介紹
Docker Compose 是一個用於定義和運行多容器Docker應用程序的工具。使用Compose，您可以通過一個{% label danger YAML %}文件來配置應用程序的服務。然後，使用一個單一的命令，創建並啟動所有服務的配置。
